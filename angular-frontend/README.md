# Sistema de Inventário - Angular

Frontend Angular que consome a API Flask.

## Instalação

\`\`\`bash
cd angular-app
npm install
\`\`\`

## Executar

\`\`\`bash
npm start
\`\`\`

Acesse: http://localhost:4200

## Estrutura

- `src/app/components/` - Componentes reutilizáveis
- `src/app/pages/` - Páginas da aplicação
- `src/app/services/` - Serviços para consumir a API
